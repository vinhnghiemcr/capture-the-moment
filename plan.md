# Website Development Plan

## Tech Stack Recommendation (Free/Cheap)
Given Nghiem's preference for free/low-cost options:

| Component | Recommendation | Cost |
|-----------|----------------|------|
| Framework | Static HTML + Tailwind CSS | Free |
| Hosting | Netlify or Vercel | Free tier |
| Images | Cloudinary or direct | Free tier available |
| Forms | Netlify Forms or Formspree | Free tier |
| Domain | Porkbun/Namecheap | ~$10/year |

## Alternative: No-Code Builders
- **Carrd.co** - $19/year, simple landing pages
- **Webflow** - Free tier available, more complex

## Proposed Pages
1. **Home** - Hero with best work, services overview
2. **Portfolio** - Categorized gallery
3. **About** - Photographer bio, equipment, approach
4. **Services/Pricing** - Packages and rates
5. **Contact/Book** - Form + booking integration
6. **Testimonials** - Client reviews (future)
