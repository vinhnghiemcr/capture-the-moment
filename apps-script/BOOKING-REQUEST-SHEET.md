# Booking Requests Sheet

Create a Google Sheet with a tab named:
- `Booking Requests`

The Apps Script will auto-write headers if the sheet is empty.

Expected columns:
1. Created At
2. Updated At
3. Request Reference
4. Status
5. Client Name
6. Email
7. Phone
8. Session Type
9. Preferred Date
10. Preferred Time
11. Backup Date
12. Backup Time
13. Location
14. School Name
15. Number of People
16. Source
17. Instagram Handle
18. Notes
19. Cancellation Note
20. Calendar Event ID
21. Customer Email Status
22. Internal Email Status
